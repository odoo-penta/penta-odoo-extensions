# PENTA Extensions - Odoo 18.0

Módulos de extensión genéricos para Odoo 18.0.

## Módulos:
- **penta_base**: Funcionalidades base
- **penta_mail_integration**: Integración email
- **penta_report_designer**: Diseñador reportes
- **penta_dashboard**: Dashboard personalizable

Este repositorio está configurado como submódulo para Odoo.sh.
